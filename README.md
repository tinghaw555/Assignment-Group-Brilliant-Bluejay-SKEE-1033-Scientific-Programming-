## 1.0 Introduction

Python is a widely used programming language in data analysis due to its simplicity, flexibility, and extensive libraries such as NumPy, Pandas, Matplotlib, and Scikit-learn. These tools allow efficient data cleaning, analysis, visualisation, and predictive modelling, making Python highly suitable for engineering and energy-related studies.

In this assignment, Python is used to analyse household electricity consumption data collected from Urban, Suburban, and Rural regions between 2018 and 2022. The dataset includes monthly electricity consumption, electricity cost, and the number of occupants per household. Through data cleaning, descriptive analysis, visualisation, and predictive modelling, this assignment aims to understand electricity usage patterns and predict electricity costs based on consumption behaviour and household size.

---

## 2.0 Objective

The aim of this assignment is to apply the data analytics knowledge learned in class to analyse household electricity consumption using a given dataset. The analysis focuses on understanding electricity usage patterns across different regions and examining the factors that influence electricity cost. In addition, a predictive model is developed and evaluated to estimate electricity costs based on consumption and household occupancy. Throughout this assignment, the objectives are to:

1. Identify missing values and data issues such as outliers and handle them using appropriate techniques.  
2. Perform descriptive statistical analysis and correlation analysis to study the relationship between electricity consumption and household size.  
3. Visualise electricity consumption trends and regional differences using line and scatter plots.  
4. Develop and evaluate a linear regression model to predict electricity cost using R-squared (R¬≤) and Mean Absolute Error (MAE).

## 8.0 Conclusion

This assignment demonstrates the practical use of Python for analysing household electricity consumption data from urban, suburban, and rural areas between 2018 and 2022. Data cleaning techniques such as interpolation and the IQR method were used to handle missing values and identify outliers, improving the quality of the dataset.

The analysis revealed differences in electricity usage across regions, with larger households generally consuming more electricity. Correlation analysis and visualisations helped identify relationships, trends, and unusual patterns. A linear regression model was developed to predict electricity costs and evaluated using R-squared and MAE, showing reasonable accuracy. Overall, the use of Python libraries such as NumPy, Pandas, and Matplotlib highlights the importance of data analysis skills in supporting forecasting and informed decision-making.
